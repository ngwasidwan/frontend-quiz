# Frontend Quiz

Project is a frontend quiz app for answering frontend related questions.

## App setup

run npm run dev to start vite.
run npm run server to start the json server.

### App Structure

An answer can be selected only ones.
If a selected answer is incorrect, then that answer is given a pale red background and the correct answer will be given a pale blue-black background simultaneously

If a selected answer is correct, then that answer is given a pale blue background

We can go to the next question only after selecting an answer

### Project objective

project was built to master the useReducer hook, understand the basics of react-router and css modules
